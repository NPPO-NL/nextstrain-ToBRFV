# Tomato brown rugose fruit virus

## Description

Tomato brown rugose fruit virus (ToBRFV) is a Tobamovirus that was first observed in 2014 and 2015 on tomato plants in Israel and Jordan respectively. Since the first description, the virus has been reported from all continents except Antarctica, and has been found infecting both tomato and pepper crops. Shows 455 genomic sequences from 32 countries from materials sampled between November 2014 and January 2025. Sequences are now trimmed relative to the coding sequence of KT383474.1
